Prosper202 Tracking202 WordPress Plugin.  Allows you to track stats to any page easily.

== Description ==

This plugin allows you to use prosper202 or tracking202 easily with WordPress.  No need to mess around with any WordPress
files.  You just upload the plugin, go into the page for the plugin, add pages and you're done.

== Installation ==

1. Upload `prosper202-tracking.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Useage ==

1. Under Tools look for Prosper Tracking Settings
2. Under Account - Name put something descriptive for example the page name or the campaign name for that page
3. Under Pages IDs put the page ID for each page that you want tracked.  To find page IDs go to Pages or Posts and then click Edit.
Mouse over the page you want to track and you will get results like example: http://customrequest.com/wp-admin/page.php?action=edit&post=172.
172 will be your page ID for this example.
4. Insert Tracking Domain URL from Prosper202 or Tracking 202.
5. Click the donate button on the plugin settings page and donate to keep development of the plugin going.
6. Relax have a beer.  Your tracking problems are over and you didn't even have to edit any WordPress files.

== Frequently Asked Questions ==

= Will this track all of my pages and campaigns? =

Yes if you include all of your pages and campaigns.

= Does this only work for Prosper202 and Tracking202? =

Who knows?  Try it out with other scripts you might be surprised and it might work with other scripts.
